# TTS And Spell

Text-to-speech spelling bee game! Words are read aloud and you type out the spelling, just like a real spelling bee!

Built with **React** on the frontend, with a **Node/Express** backend both using  **TypeScript** throughout. Text-to-speech is powered by the **ElevenLabs** API.

## Tech Stack

- React + TypeScript (Vite)
- Node.js + Express
- ElevenLabs TTS API
- SCSS Modules for styling
